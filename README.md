# Papyrus

Papyrus is a web-based academic management system focused on **faculty access and attendance workflow**.  
This repository follows a clean separation of **frontend**, **backend**, and **database** layers and includes a live **frontend-only deployment** for demonstration purposes.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP (session-based)
- Database: MySQL, Local browser storage
- Hosting: Vercel (frontend)

---

## Features
- Faculty as well as Student login & signup UI
- Attendance system interface
- Protected navigation flow 
- Schedule planner & viewer interface
- Clean, modular project structure

---

## Running Locally
1. Clone the repository
2. Serve `frontend/` using a local server
3. For full-stack usage, configure PHP and import `database/schema.sql`

---

## Deployment
- Frontend is deployed as a static site on **Vercel**
- Backend can be deployed separately on a PHP-supported host
